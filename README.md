# Don't cunning! Think.

## セットアップ

node v18.17.1 を使用します  
※ 歩数計を動作させるには、加速度センサーを搭載したAndroidデバイスを用意してください

```bash
$ git clone https://github.com/uyupun/dcf-app.git

$ cd dcf-app

$ npm i

$ npm run dev
```
